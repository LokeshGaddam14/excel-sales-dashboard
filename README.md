# E-Commerce Excel Sales Dashboard

## Overview
Interactive Excel dashboard analyzing 9,994 orders
with pivot tables, slicers and KPI cards.

## Tools
Microsoft Excel, Pivot Tables, PivotCharts, Slicers

## Key Insights
1. Technology leads revenue at $836K with 17% margin
2. West region most profitable at $108K
3. Q4 shows consistent 40% seasonal sales spike
4. Top 10 customers drive outsized revenue
5. Furniture margin critically low at 7%
